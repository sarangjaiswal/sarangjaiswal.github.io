How to Set JAVA_HOME in MacOS

Posted on December 26, 2018 in Java • 1 min read

In this article we will discuss on how to set JAVA_HOME in MacOS

Step 1: Add Java path to .bash_profile file

Open .bash_profile file

SarangMac:~ sarangjaiswal$ vi .bash_profile

Add the following details to the file. ((Note: the JAVA_HOME path should be the path where you have installed Java))

# Setting PATH for JAVA
export JAVA_HOME=$(/usr/libexec/java_home)

Step 2: Test Java Installation

Close the terminal and open it again and fire below command to check if the installation was successful

SarangMac:sarangjaiswal.github.io sarangjaiswal$ java -version
java version "1.8.0_191"
Java(TM) SE Runtime Environment (build 1.8.0_191-b12)
Java HotSpot(TM) 64-Bit Server VM (build 25.191-b12, mixed mode)

SarangMac:~ sarangjaiswal$ echo $JAVA_HOME
/Library/Java/JavaVirtualMachines/jdk1.8.0_191.jdk/Contents/Home
