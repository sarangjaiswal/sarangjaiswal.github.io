Install Maven on MacOS

Posted on December 26, 2018 in Maven • 2 min read

In this article we will go over Maven installation on macOS.

Step 1: Ensure JDK in already installed on your system

Ensure JAVA_HOME environment variable is set and points to your JDK installation

SarangMac:Java sarangjaiswal$ echo $JAVA_HOME
/Library/Java/JavaVirtualMachines/jdk1.8.0_191.jdk/Contents/Home

Step 2: Download Maven

Download Maven from here for example apache-maven-3.6.0-bin.tar.gz and extract the zip file at your desired location (preferrably under Library folder) using following command

SarangMac:Downloads sarangjaiswal$ tar -xvf apache-maven-3.6.0-bin.tar.gz

Step 3: Add the Maven path to .bash_profile file

Open the .bash_profile

SarangMac:~ sarangjaiswal$ vi ~/.bash_profile

Add the following details to the file. (Note: the M2_HOME path should be the path where you have extracted the Maven)

#Setting PATH for MAVEN
export M2_HOME=/Library/Maven/apache-maven-3.6.0
export PATH=$PATH:$M2_HOME/bin

Step 4: Test Maven Installation

Close the terminal and open it again and fire below command to check if the installation was successful

SarangMac:~ sarangjaiswal$ mvn -version
