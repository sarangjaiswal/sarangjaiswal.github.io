Pelican - Static Site Generator (Part 1)

This is a quick tutorial on how to create a Static Website using Pelican.

What is a Pelican?

Pelican is a open source static website generator that supports Markdown or restructured Text. Its based on Python. More details on Pelican can be found here

I'll introduce you to the basics of Pelicann. If you already know Pelican, and want to jump straight to learning about the fancier things, I suggest you skip this blog post and look for other articles on Pelican on my Blog.

Also, I will be demonstating on MacOS, but the steps are more or less same for other popular platforms like Windows and Linux.

Let's jump right in.

Quick Start

Create a blank project

Create a folder where and name it appropriately for your site and switch to that directory. User pelican-quickstart to create project skeleton. This command will help you create a new Pelican-based website.

It will ask you questions on how you would like to configure your website. Once you have answered these questions it will create the project skeleton accordingly.

To keep it simple, I will be answering No to most of the questions. We will looks at more advanced examples later.

SarangMac:Examples sarangjaiswal$ pelican-quickstart 
Welcome to pelican-quickstart v4.0.1.

This script will help you create a new Pelican-based website.

Please answer the following questions so this script can generate the files
needed by Pelican.


> Where do you want to create your new web site? [.] .
> What will be the title of this web site? SimplePelicanWebSite
> Who will be the author of this web site? Sarang Jaiswal
> What will be the default language of this web site? [en] 
> Do you want to specify a URL prefix? e.g., https://example.com   (Y/n) n
> Do you want to enable article pagination? (Y/n) n
> What is your time zone? [Europe/Paris] US/Central
> Do you want to generate a tasks.py/Makefile to automate generation and publishing? (Y/n) n

Generate site

Navigate back to your site directory and run pelican to generate your website.

SarangMac:Examples sarangjaiswal$ pelican -s pelicanconf.py content/ -o output/
Done: Processed 1 article, 0 drafts, 0 pages, 0 hidden pages and 0 draft pages in 0.12 seconds.

See below for details on arguments of pelican command

  -o OUTPUT, --output OUTPUT
                        Where to output the generated files. If not specified,
                        a directory will be created, named "output" in the
                        current path. (default: None)
  -s SETTINGS, --settings SETTINGS
                        The settings of the application, this is automatically
                        set to pelicanconf.py if a file exists with this name.
                        (default: None)

Above command will convert your Markdown (.md) file into html. All the converted files will be placed in output directory.

Preview site

Navigate back to the parent folder of the site and run 

SarangMac:Examples sarangjaiswal$ pelican -l content/ -o output/ -s pelicanconf.py -p 8082

See below for details on arguments of pelican command

  -t THEME, --theme-path THEME
                        Path where to find the theme templates. If not
                        specified, it will use the default one included with
                        pelican. (default: None)
  -o OUTPUT, --output OUTPUT
                        Where to output the generated files. If not specified,
                        a directory will be created, named "output" in the
                        current path. (default: None)
  -s SETTINGS, --settings SETTINGS
                        The settings of the application, this is automatically
                        set to pelicanconf.py if a file exists with this name.
                        (default: None)

  -p PORT, --port PORT  Port to serve HTTP files at. (default: 8000) (default:
                        None)

Open the browser and give localhost:8082 to see your website.
